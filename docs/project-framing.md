# CamilleAubert.com — Project Framing Document

## 1. Project Overview

### 1.1 Project Name
**CamilleAubert.com**

### 1.2 Project Purpose
The purpose of this project is to create a premium professional portfolio website for Camille Aubert.

This website is designed to:
- establish a strong and memorable professional presence,
- present Camille Aubert as an **AI Solutions Architect** with a solid full-stack engineering background,
- serve as a high-quality professional landing page accessible from LinkedIn and other public profiles,
- centralize key professional links and future technical projects,
- support internal visibility within Cegos, especially toward decision-makers,
- provide a strong long-term foundation for future side projects and subdomains.

### 1.3 Strategic Goal
The main goal of the website is not to act as a generic developer portfolio.

Its mission is to project:
- seniority,
- precision,
- technical depth,
- design sensitivity,
- mastery of modern AI-driven software delivery.

The site must communicate, in less than one minute, that Camille Aubert is:
- serious,
- advanced,
- technically credible,
- future-facing,
- product-minded,
- operating at a high strategic level.

## 2. Product Vision

### 2.1 Vision for V1
Version 1 is a premium one-page portfolio website written in English.

Its purpose is to act as:
- a professional showcase,
- a personal brand anchor,
- a polished digital business card,
- a memorable entry point for colleagues, decision-makers, recruiters, and professional contacts.

V1 must remain focused:
- no blog,
- no lab,
- no complex multi-project platform,
- no heavy application logic,
- no Travel Planner,
- no over-engineered infrastructure.

### 2.2 Vision for the Future
In the future, the website may evolve into a broader personal platform including:
- subdomains for side projects,
- technical playgrounds,
- fast experimental apps,
- a blog,
- richer technical demonstrations.

This future vision is important, but explicitly out of scope for V1.
