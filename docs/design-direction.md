# Design Direction

## Design objective

The website must create an immediate premium impression.

The design should communicate:
- seriousness,
- expertise,
- elegance,
- technical clarity,
- modernity.

## Visual principles
- sober
- premium
- mobile-first
- minimal but striking
- smooth and polished
- visually controlled

## Current preferences
- Typography: **Satoshi**
- Style: premium tech
- Layout: likely Bento-inspired
- Footer: minimal
- Navigation: simple anchor-based scroll
- Icons: small, refined, carefully chosen
- Motion: smooth, elegant, restrained

## Avoid
- noisy effects
- overloaded pages
- fake startup aesthetics
- training-sales vibes
- geeky UI clichés
- visual clutter
