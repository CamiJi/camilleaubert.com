# camilleaubert.com

Main repository for the `camilleaubert.com` portfolio website.

## Purpose

This repository contains the main portfolio website for Camille Aubert.

It is the central repository for:
- the Astro application,
- product and content framing,
- design direction,
- project-level documentation,
- VSCode workspace configuration,
- CI/CD workflows for the website.

## Related repositories

This project is part of a small multi-repository setup.

### Main repositories
- **Portfolio repository:** `CamiJi/camilleaubert.com`
- **Infrastructure repository:** `CamiJi/camilleaubert-infra`

### Repository responsibilities

#### `camilleaubert.com`
Contains:
- website source code,
- product framing,
- content and design documentation,
- local development setup,
- VSCode workspace files,
- website deployment workflows.

#### `camilleaubert-infra`
Contains:
- infrastructure audit,
- DNS and SSL documentation,
- reverse proxy documentation,
- deployment runbooks,
- server operations notes,
- future subdomain routing strategy.

## Product goal

The first version of `camilleaubert.com` is a premium professional portfolio website designed to:
- establish a strong personal brand,
- present Camille Aubert as an AI Solutions Architect,
- create a polished digital business card,
- support internal visibility inside Cegos,
- provide a future foundation for side projects and subdomains.

## Current scope

V1 focuses on:
- one main portfolio website,
- English content,
- strong mobile-first experience,
- premium and clean design,
- simple deployment model,
- lightweight technical architecture.

Out of scope for V1:
- blog,
- lab,
- active side-project subdomains,
- complex multi-app architecture,
- Travel Planner.
